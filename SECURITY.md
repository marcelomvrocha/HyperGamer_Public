# Security Policy

## Supported Version

Security fixes are applied to the current public standalone release.

## Reporting a Vulnerability

Do not post exploit details in a public issue.

For sensitive reports, use the contact form at:

- https://www.mmrokku.com/contact.html

Include:

- a short summary of the issue
- affected route, page, or component
- reproduction steps
- impact assessment
- any suggested mitigation if you already have one

If a private response path is added later, this document should be updated to point to it directly.
